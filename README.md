## NetDiag — Network Diagnostic Tool for iOS

NetDiag is a powerful network diagnostic tool that includes:
- Ping
- Traceroute
- NSLookup
- Whois
- LAN Scanner

## Privacy Policy

NetDiag does not collect or share any personal data.
The app does not require registration or login.
All network requests are made directly from your device.
